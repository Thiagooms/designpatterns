Desafio — Case 3

Smell Code para refatorar

Contexto

Uma plataforma de e-commerce cresceu rápido e o desenvolvedor original foi adicionando canais de notificação diretamente no serviço principal conforme a necessidade surgia.

O código funciona, mas virou um problema: toda vez que um novo canal precisa ser adicionado ou alterado, é necessário mexer na classe ServicoNotificacao — e o risco de quebrar algo já existente é alto.

Seu trabalho é identificar o problema e refatorar o código aplicando o padrão correto.

Sobre o smell code

Switch Statements + Inappropriate Intimacy

O Switch Statements (ou cadeias de if/else if) é um smell code clássico descrito por Martin Fowler no livro Refactoring. Ele aparece quando uma classe precisa decidir qual objeto criar ou qual comportamento executar com base em um tipo ou string — e essa decisão cresce a cada novo caso adicionado.

Combinado com o Inappropriate Intimacy: ServicoNotificacao conhece detalhes internos de todas as implementações concretas (NotificacaoEmail, NotificacaoSMS, NotificacaoPush), criando um acoplamento que não deveria existir nessa camada.

O sintoma mais direto é o new dentro do if: sempre que a criação de um objeto está escondida dentro de uma condicional, é sinal de que a responsabilidade de criação está no lugar errado.

Código atual

smell code 
public class ServicoNotificacao {

    public void notificar(String canal, String destinatario, String mensagem) {

        if (canal.equals("email")) {
            NotificacaoEmail n = new NotificacaoEmail();
            n.enviar(destinatario, mensagem);

        } else if (canal.equals("sms")) {
            NotificacaoSMS n = new NotificacaoSMS();
            n.enviar(destinatario, mensagem);

        } else if (canal.equals("push")) {
            NotificacaoPush n = new NotificacaoPush();
            n.enviar(destinatario, mensagem);

        } else {
            throw new IllegalArgumentException("Canal desconhecido: " + canal);
        }
    }
}

public class Main {

    public static void main(String[] args) {
        ServicoNotificacao servico = new ServicoNotificacao();

        servico.notificar("email", "thiago@email.com", "Seu pedido #4521 foi confirmado!");
        servico.notificar("sms",   "+5511999990000",  "Seu pedido #4521 foi confirmado!");
        servico.notificar("push",  "thiago@email.com", "Seu pedido #4521 foi confirmado!");
    }
}
