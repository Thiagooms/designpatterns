Desafio — Case 6

Smell Code para refatorar

Contexto

Uma corretora de investimentos calcula o imposto sobre o lucro de acordo com o tipo de investimento do cliente: Renda Fixa, Ações ou Fundos Imobiliários. Cada tipo possui uma alíquota e uma regra de cálculo diferente.

O time quer adicionar o tipo Criptomoedas na próxima versão, com alíquota flat de 15% sem nenhuma isenção.

Sobre o smell code

Switch Statements

O CalculadoraDeImposto concentra as regras de cálculo de todos os tipos de investimento. Cada vez que uma nova regra fiscal surge ou um novo tipo é adicionado, essa classe precisa ser aberta e modificada — quebrando o princípio Open/Closed.

Código atual

smell code 
public class CalculadoraDeImposto {

    public double calcular(String tipo, double lucro) {

        if (tipo.equals("RENDA_FIXA")) {
            // Alíquota regressiva: quanto mais tempo, menor o imposto
            // Simulação: 22,5% até 180 dias — aqui simplificado como fixo
            double imposto = lucro * 0.225;
            System.out.println("Renda Fixa — alíquota 22,5%");
            System.out.println("Imposto devido: R$" + imposto);
            return imposto;

        } else if (tipo.equals("ACOES")) {
            // Isenção para vendas abaixo de R$20.000 no mês
            if (lucro <= 20000) {
                System.out.println("Ações — lucro abaixo de R$20.000, isento de IR.");
                return 0;
            }
            double imposto = lucro * 0.15;
            System.out.println("Ações — alíquota 15%");
            System.out.println("Imposto devido: R$" + imposto);
            return imposto;

        } else if (tipo.equals("FII")) {
            // Fundos Imobiliários: 20% sobre lucro, mas dividendos são isentos
            double lucroTributavel = lucro * 0.7; // 30% são dividendos isentos
            double imposto = lucroTributavel * 0.20;
            System.out.println("FII — 20% sobre 70% do lucro (30% isentos como dividendos)");
            System.out.println("Imposto devido: R$" + imposto);
            return imposto;

        } else {
            throw new IllegalArgumentException("Tipo de investimento desconhecido.");
        }
    }
}
Cada tipo de investimento tem sua própria regra fiscal: Renda Fixa aplica alíquota fixa, Ações tem isenção abaixo de R$20.000, e FIIs tributam apenas 70% do lucro. Como isolar cada algoritmo de cálculo sem abrir a CalculadoraDeImposto toda vez que uma nova regra ou tipo surgir?

A solução deve usar uma única interface de estratégia: double calcular(double lucro). O CalculadoraDeImposto recebe a estratégia pelo construtor e delega o cálculo sem conhecer a regra interna de nenhum tipo.
